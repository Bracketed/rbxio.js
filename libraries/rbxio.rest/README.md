# @bracketed/rbxio.rest

REST API wrapper for Roblox APIs, including Cloud and Web APIs.

## Installation

```sh
# Using yarn
yarn add @bracketed/rbxio.rest

# Using npm
npm install @bracketed/rbxio.rest
```
